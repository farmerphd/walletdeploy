CLI Quick Reference

Run these commands before using WalletDeploy. Replace addresses with your own.

Set Buffer Authority

Transfer buffer authority to your cold wallet before upgrading.

Recover SOL from Buffer

Close an orphaned buffer and recover locked SOL.

Transfer Upgrade Authority

Move program upgrade authority to any address.

Extend Program

Extend ProgramData before upgrading to a larger binary.

Deploy Program

Deploy a new program with your cold wallet as upgrade authority from block 0. Write the buffer via CLI first, then deploy here. Note: The program keypair (one-time use) must also sign — generate a throwaway keypair with solana-keygen new -o program-keypair.json, use it once to deploy, then discard it. Your cold wallet becomes the permanent upgrade authority.

Upgrade Program

Execute upgrade after writing buffer from CLI and setting buffer authority above.
